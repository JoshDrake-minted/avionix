Installation
============

.. _kubernetes: https://kubernetes.io/
.. _helm: https://helm.sh/docs/intro/install/

Before installing avionix you must also have a kubernetes_
cluster and helm_ setup for that cluster.

kubectl_ is also recommended.

-----

.. code-block:: bash

    pip install avionix
